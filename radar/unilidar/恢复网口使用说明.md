# 宇树 L2 恢复网口 ENET UDP 使用说明

当前如果雷达已经被切到 Type-C / TTL UART 输出，恢复网口需要做两件事：

1. 通过串口给雷达下发 `work_mode=0`，让雷达切回 ENET UDP 输出。
2. 把 ROS2 launch 配置从串口模式改回 UDP 模式。

## 1. 通过 Type-C 串口把雷达切回网口输出

接线保持为：

- 雷达电源口接 `12V/1A` 电源
- 雷达 `4PIN 串口` 接宇树转接模块
- 转接模块 `Type-C` 接电脑

确认串口存在：

```bash
ls -l /dev/ttyACM0
sudo chmod a+rw /dev/ttyACM0
```

用 ROS 节点直接通过串口下发 `work_mode=0`：

```bash
source /home/linkchen/lidar/radar/unilidar/unitree_lidar_ros2/install/setup.bash

ros2 run unitree_lidar_ros2 unitree_lidar_ros2_node --ros-args \
  -p initialize_type:=1 \
  -p work_mode:=0 \
  -p serial_port:=/dev/ttyACM0 \
  -p baudrate:=4000000
```

启动后等待 `2-3` 秒，然后按 `Ctrl+C` 停止。

然后给雷达断电重启。

## 2. 把 ROS2 launch 改回网口模式

执行下面命令，把源码和已安装的 launch 都改回 UDP：

```bash
cd /home/linkchen/lidar/radar/unilidar

perl -0pi -e "s/'initialize_type': 1/'initialize_type': 2/g; s/'work_mode': 8/'work_mode': 0/g" \
  unitree_lidar_ros2/src/unitree_lidar_ros2/launch/launch.py \
  unitree_lidar_ros2/install/unitree_lidar_ros2/share/unitree_lidar_ros2/launch.py
```

确认配置：

```bash
grep -nE "initialize_type|work_mode|lidar_ip|local_ip|lidar_port|local_port" \
  /home/linkchen/lidar/radar/unilidar/unitree_lidar_ros2/install/unitree_lidar_ros2/share/unitree_lidar_ros2/launch.py
```

应看到类似：

```text
{'initialize_type': 2},
{'work_mode': 0},
{'lidar_port': 6101},
{'lidar_ip': '192.168.1.62'},
{'local_port': 6201},
{'local_ip': '192.168.1.2'},
```

## 3. 配置电脑网口 IP

雷达默认配置：

- 雷达 IP：`192.168.1.62`
- 电脑 IP：`192.168.1.2`
- 雷达发送端口：`6101`
- 电脑接收端口：`6201`

先查看网卡名：

```bash
ip link
```

假设有线网卡名是 `enp3s0`，配置电脑 IP：

```bash
sudo ip addr flush dev enp3s0
sudo ip addr add 192.168.1.2/24 dev enp3s0
sudo ip link set enp3s0 up
```

如果你的网卡名不是 `enp3s0`，把命令里的 `enp3s0` 替换成实际网卡名。

## 4. 验证 UDP SDK 有数据

接线改为：

- 雷达电源口接 `12V/1A` 电源
- 雷达 `RJ45` 网口接电脑或交换机

运行官方 UDP 示例：

```bash
cd /home/linkchen/lidar/radar/unilidar/unitree_lidar_sdk
./bin/example_lidar_udp
```

如果恢复成功，会看到点云或 IMU 解析输出，例如：

```text
A Cloud msg is parsed!
An IMU msg is parsed!
```

## 5. 启动 ROS2 网口模式

```bash
source /home/linkchen/lidar/radar/unilidar/unitree_lidar_ros2/install/setup.bash
ros2 launch unitree_lidar_ros2 launch.py
```

## 常见问题

如果 SDK UDP 示例没有数据：

- 确认雷达已经断电重启。
- 确认网线接的是雷达 `RJ45`，不是 Type-C 转接模块。
- 确认电脑 IP 是 `192.168.1.2/24`。
- 确认没有其他程序占用 UDP `6201` 端口。
- 如果仍无数据，可先接回 Type-C，再重复第 1 步下发 `work_mode=0`。
